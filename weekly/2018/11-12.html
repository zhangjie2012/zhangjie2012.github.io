Vagrant is for managing machines, and Docker is for building and running application environments.

最简单的讲 VM 和虚拟机的区别，就是 VM 是针对的虚拟机环境，而 Docker 针对的是应用的构建和运行环境。 也就是说前者针对的是 OS，Docker 针对的是 App。
