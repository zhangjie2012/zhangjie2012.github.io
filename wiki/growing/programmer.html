1 办公硬件

  • 耳机
    • Grado SR325e：纯手工制作耳机，长相帅的一B，就是包装丑了一点，当初是为了听摇滚买的。
    • 你也可以买森海塞尔，比较中庸（听什么都 OK），我后来买了 森海塞尔（Sennheiser）MOMENTUM In-Ear I Black Chrome，只能说电子产品一分钱一分货，相比 Grado 还是有一定差距的
  • 机械键盘
    • FILCO 87 圣手二代 茶轴，千元左右
    • Cherry MX2.0 G80-3800 性价比超级高，不喜欢机械键盘的朋友也可以尝试这款的低键位
  • 显示器
    • DELL U2718Q 27 英寸 4K
    • 非 MacOS，2K 屏幕也可
  • 鼠标
    • Logitech M558 无线鼠标：小巧好用的蓝牙无线鼠标，支持简单的快捷键定制
  • 笔记本
    • NVV N-W1：非常好用的笔记本支架，折叠完了之后跟扇子差不多大小

说明：

  • 对于耳机，入耳式和头戴式看个人喜好，两个切换着用效果更佳；蓝牙耳机，会有一定的音质损伤，看怎么权衡了
  • 对于显示器，我自己测试下来，Mac mini 刷新率跟不上 4K，所以只能买 2K；但是对于笔记本，最新的系统 10.15.1 2K 与 4K 的差距还挺大的，10.15 之前没这种感觉
  • 对于机械键盘，我买过 2 个 Cherry 的，2 个 Filco 的，还有一个 IKBC 的。轴用过青轴和茶轴，还有朋友的红轴。结论供参考：
    1. 原厂轴太重要了，IKBC 算是国内做的比较好的了，用的时间长了，轴跟 Filco 的还是有一定差距的（按着手指头累）
    2. Cherry 键盘普遍比 Filco 丑，真的丑
    3. 键盘不建议买奶白色，漂亮是绝对漂亮，就是太不禁脏了，别听信洗键帽之说，我有两个键盘洗过键帽，装回去之后总感觉怪怪的
    4. 推荐茶轴，青轴不适合写代码，一来太吵，而来确实累；红轴太轻了，几乎没有任何机械键盘的快感，还是算了
    5. 机械键盘用的时间长了，还是会比较累（难道是因为年龄大了？），后面准备低键位的红轴了

2 编码与环境

  • IDE/Editor 我不用 IDE
    • Emacs，主力编辑器，我的 Wiki
    • Vim，没办法，服务器只有 Vi/Vim
  • 编码字体
    • Fira Code
    • Source Code Pro
    • 更纱黑体 中英文等宽等高，好评，Microsoft-Yahei-Mono 和 文泉驿也可以，但是太丑
    • Jetbrains Mono 官方推荐 13 号字体和 1.20 倍行号
  • 包管理工具
    • Homebrew：The missing package manager for macOS
  • 版本管理
    • git
  • 命令行使用优化
    • ag
    • autojump：目录快捷跳转
    • tmux：终端复用
  • SSH 连接工具 & 终端模拟器
    • iterm2
    • zsh + oh-my-zsh
    • XShell：可惜只支持 Windows 环境
  • Chrome 插件
    • Postman：也可独立于 Chrome 运行
    • FireShot：网页截图
    • SwitchyOmega：你懂的
  • 代码生成图片
    • carbon
  • 文本比较
    • Beyond Compare，跨平台，注册机
  • 版本号约定
    • semver: <MAJORL>.<MINOR>.<PATCH> ，大部分软件发布都是用的这种规则
    • calver: YY.mm.<patch> ，基于日历的版本发布，适用于对时间敏感的项目，目前使用这种规则的有 Docker、Ubuntu 等
  • 思维导图
    • XMind，WPS 也不错（但是 Linux 下的 WPS 不支持）。
  • UML
    • Mac 平台下的 WPS 支持画图，很好用
    • Windows 下当然是 Visio 啦，各种在线的都感觉不好用
  • Docker

3 工具软件

3.1 公有云

目前只推荐腾讯云，UCloud 也还不错毕竟小厂。阿里云就算了，坑太多了。

3.2 记录

  • 云笔记
    • 有道云笔记，支持 Windows 和 Mac 平台，Mac 体验优于印象笔记，不支持 Ubuntu，手机上广告很多。 开始使用 org-mode 之后，云笔记也不咋用了。
  • TODO
    • Wunderlist：支持 Windows 和 Mac 平台，Ubuntu ，部分功能用 org-mode 代替了。 gtd 是一个很艰难的事，与工具无关。

3.3 邮箱客户端

  • 网易邮箱大师：支持 Mac 和 Windows，不支持 Linux

说明：

  • 用了 Foxmail 几年时间，最后弃了，主要原因是现在已经没人维护了，MacOS 下邮件接收太猛的时候（告警邮件），邮件会掉线，再登录一直提示密码错误，这个问题很严重
  • Mailspring：长相不错，跨全平台（Windows/MacOS/Linux），轻量级，就是 Bug 有点多
