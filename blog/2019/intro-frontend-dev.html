3.1 前端开发经历

2014 之前 - 随便玩玩
WordPress 主题（静态页面） => CSS / HTML
2014 ~ 2017 - 生产化（数据统计平台，ToB）
前后端不分离（SSR） => JS/CSS/HTML， bootstrap，webpack
2018 ~ 2019 - 企业中后台
前后端分离（SPA，CSR） => ES6/ LESS /HTML，Ant Design Pro（React 全家桶 + antd）

3.3 总结

  • 前端技术是 产品化 必不可少的一个环节（尤其对于个人开发者）
  • 前端技术是相通的，掌握其中一个端，其他的触类旁通
  • 了解一点前端开发模式，可以更好的前后端协作
