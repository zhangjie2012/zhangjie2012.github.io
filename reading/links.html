3. DevOps

侧重于企业级的工具、架构、DB 等。

  • Ansible: 自动化运维工具
  • confd: 轻量级的配置管理工具
  • Docker：容器
  • ELK: Elasticsearch, Logstash, Kibana
  • Gitlab：代码管理，CI/CD，DevOps
  • Grafana: 数据可视化，监控支持 Graphite, InfluxDB, Prometheus, ES 等多种数据库
  • influxDB：时间序列 DB
  • Kafka: 开源的高性能的分布式的流式平台，一般用作消息中间件
  • Kubernetes: 生产级别的容器编排工具
  • Prometheus: 开源监控告警解决方案集成信息收集入库、存储，展示和告警的开源监控告警解决方案
  • salt: 自动化运维工具
  • TICK: Telegraf + InfluxDB + Chronograf + Kapacitor 组合工具栈

4. 数据库

  • MongoDB: 开源的文档式数据库，存储灵活（BSON，类 JSON）
  • MySQL: 关系型数据库
  • Redis: NoSQL，开源的内存结构化存储的数据库，常作为数据库，高速缓存和消息队列
  • TiDB: 开源的分布式关系型数据库，完全兼容 MySQL 协议
