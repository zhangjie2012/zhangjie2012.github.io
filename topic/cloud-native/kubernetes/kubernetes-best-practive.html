5. 使用技巧

5.1. 使用 Kubernetes 最常犯的错误

1、Liveness 和 Readiness 探针：

  1. Liveness 探针失败，自动杀掉重新启动
  2. Readiness 探针失败，从负载均衡器的 endpoints 中移出

慎用 Liveness，有时候 Pod 在处理任务 hang 住是比较正常的。Readiness 在一个服务包含多个 pods 时建议用起来。
