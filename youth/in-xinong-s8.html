在西农-S8 技术、论坛、博客、社交(下)

昨天本应写 技术、论坛、博客、社交(下) 的，结果写着写着跑偏了，索性直接更新了一篇「鸡汤」。

在这 7 年里，我认识的朋友中，比较佩服的人有两个：一个是 TankyWoo，一个是陈东坡。不是因为他们牛逼，也不是因为他们帮过我很多，只是因为他们做了我曾经想做而没有做的事。陈东坡在后面的文章再写，这篇先讲 Tanky。

《在西农-S6》中挖了下面的坑：

  • Tanky 到底何许人也？
  • 我是如何与 Tanky 相识、相知的？
  • 一个写明明可以在程序设计、算法领域有所作为的男人，为何偏偏去做了搬砖的？
  • 一个当年意气风发，舌战群儒的喷子，为何现在只会静静的抽着眼，默默的装着B，并且在装B这条路上渐行渐远？

明显有些托大了，但是自己挖的坑只能自己埋。只能按照我对他的了解来写写了（对于别人，我需要征求他们是否同意我写他们的故事，而 Tanky 是不必要的，他是我最要好的几个朋友之一，更重要的是他喜欢这种存在感）。

TankyWoo 这个 ID 是从他的名气是音译过来的，我很少直呼他的名字，即便是在我们自己的群里，这么多年就这么叫过来了，改不了了。

Tanky 是 北化自动化系 的「高材生」，非计算机相关专业。出于对计算机的憧憬，所以打算学习计算机编程。也不知道怎么开始，琢磨着总得买几本书看吧，于是就搞了基本 Photoshop, FLASH, Dreamweaver 的书开始学习编程。 过了一段时间以后发现哪里好像不太对，可能自己审美上的硬伤，总也弄不好，就放弃了这个方向（还好及时收手）。

后来一问才知道，妈蛋，人家搞计算机是要学习 C 语言的，要不我也学C语言。学了之后，诶？好像突然感觉对了，编程应该就是这种感觉吧，没错，一定是的！买书！之后花了 2000 大洋买了一大摞计算机程序相关的书籍。 我相信和他和很多人一样，开始这条路都是因为那么一点快感，而慢慢的喜欢程序，然后又因为喜欢，才越做越好。那些非科班的人能够走到编程这个领域，除了比其他人付出更多以外，更重要是他们拥有浓重的兴趣。

一个人学习总是寂寞的。在 2009 年的时候，Tanky 在网上找了一堆搞 C++ 的小菜鸡，建了一些群。随着群越来越多，管理上也越来越困难。其实管理成本只是一方面，还有一方面技术群做着做着就逐渐成了扯淡群，沟通效率极低。 于是他就有了创办了自己论坛讨论技术的想法。

当时在学 C++，可能觉得这条路可能很长，需要自己坚持不懈的努力。没过多久，一个叫 「C++奋斗乐园」的论坛就出现了（这实在是一个没有创意的名字）。

又没过多久，不经意间，Tanky 接触到了 ACM，从此一发不可收拾，开始了自己两三年的疯狂刷题生涯。

起初，论坛的热度远远比不上群。这是有历史原因，Tanky 是先做的群，然后才做的论坛；有了论坛以后，就出现了论坛和群共存的尴尬情况，很多新人进了论坛以后进群就再也没来过论坛。我记得我刚去论坛的时候，群号已经排到十几了，一个群有个上百号人，也算小有规模了。 尽管后来 Tanky 努力的引导小菜鸡们去论坛里讨论问题，但实际上能从群转到论坛的还是少之又少。最后，个人精力有限等等因素，十几个群七零八落。现在印象中只剩下一个 「[奋斗乐园]职业交流群」了。

再起初，论坛是打着「C++奋斗乐园」的旗号，却干着大家一起刷题的活。也就说，这并不是一个严格意义的论坛，只是一个大群搞算法人的研究算法的地方。相当于 Tanky 搞了一个 C++ 论坛，然后大家去愉快的刷题啦。

直到，有一天，一个叫 JerryZhang（那个时候好像还是 chinazhangjie）的男人出现在论坛以后，以他高超的技艺征服了各位小菜鸡，坐镇 C++ 板块，C++ 板块才逐渐有了生机 。自此，算法与语言一时成了 Tanky 口中的「掎角之势」。:-D

2011 ~ 2012 应该是论坛发展最好的两年了，这是为什么呢？

  • 时间：大学生最宝贵的和最充裕的可能就是时间了。那时大学的后两年时间，都熬成了老油条，像我和 Tanky 这种基本上早就远离课堂了；
  • 兴趣：我前面说过很多次，入门的时候，能让自己有征服的快感和别人给予的肯定是很重要的。当有问题的时候，通过自己的尝试和验证最终得到一个自己认为最精确的答案，这本身就是很酷的一件事情。何况还有别人的赞美。我和 Tanky 有一点是很相似的，当我们得到别人夸奖的时候，不谦让也不回复，虽然心里喊着还想要，但是还是会一个保持高冷的大神姿态。Tanky 的话，还得默默的点上一支烟，享受几分钟。
  • 水平低、能力弱、视野窄：不管上面怎么吹嘘，那个时候的我们真的很菜。因为自己很菜还想把论坛做好，因为面对的都是不同的问题，你都要去解决，所以只能付出更多的时间在上面。
  • 朋友：论坛里的版主和一部分人，都是跟着论坛一起过来的，大家基本上每天都在线。一起研究问题啦、博客加个友链啦、一起去喷个人啦~

从 13 年之后，论坛都开始衰落了，这是又为什么呢？因为上面的所有「优势」随着大家毕业工作都不复存在了。

用一句话概括：视角变了。

工作以后面临的问题比大学要多的多，要适应一个新的生活、工作环境。尤其是刚参与工作的时候，新的技术、新的工具都要去学习。以前刷题的，刷不了了；以前玩 C++ 的，可能去做了 Java；有人去做开发；有人去做测试；Tanky 去做了运维；甚至有人都不从事这个行业了。 大多数人这些事情都应接不暇呢，哪有人管的了论坛的死活。

对于论坛的管理者 Tanky，他对论坛的态度从 一大堆人一起玩刷的地方 变成了 我要把它变成一个「优秀的技术论坛」。那些小菜的问题，早就不屑于去解决了。论坛提供的是一个平台，一个桥梁，让它能有自我治愈的能力（有人进来，有人离开，保持活跃），而不是事事都参与进去。 从 2010 年到 2012 年这三年的时间，我一直担任的是一个活跃用户的角色，2013 年到北京工作，才逐渐成了管理者。

我们一心想把它做成一个优秀的技术论坛，想了很多办法，但最后也没能改变什么，就这样勉强撑到了 2014 年。我离开了北京，去上海了；Tanky 一次误操作把论坛 DB 删了（干的漂亮），妈蛋，这次彻底没了……

论坛对于 Tanky 的意义不言而喻，如果没有论坛可能就不会有现在的Tanky。我个人理解，论坛给予他这些：

  • 第一、技术成长 。不说在论坛里面学到的知识，光论坛部署、维护、管理 这一套东西都够折腾的了。如果不是要搞论坛，我想他不会那么早的接触 Linux ，要知道一旦开始折腾 Linux ，要面对的是整套系统，或者说一个全新的技术世界。如果不是那么早接触 Linux ，那他现在还靠什么搬砖。
  • 第二、人脉。在几年的论坛折腾中，结交了很多朋友，这些人分布在各大公司，他们可都是未来互联网的希望啊（比如那个叫Jerry的）。性格原因吧，Tanky 在人脉的处理上和个人品牌的打造上，还挺值得很多人去学习。要说个人魅力，他还是有一点的。
  • 第三、一段青春。我以前以为每个人都是有青春的，后来我发现我错了，很多人对大学、对过去似乎没什么回忆。能有一段自己值得回忆的青春，这才是最宝贵的财富。不然，拿什么出去跟人吹牛逼啊。

我们抛开人为感情因素谈奋斗乐园，可能从一开始就决定了论坛不可能变成我们期望的那样，充其量就是一个玩物。这种一开始就没想明白、边做边看的产品，不可能走的很远的。大部分产品的最后的死就是死在开始没方向，随心去做，越做越迷茫，最后做不下去了。 「C++奋斗乐园」是这样、「乐魂」也是这样。再者，在 Tanky 的大学成长的阶段做这样的一款产品，注定是要失败的。

再抛开奋斗乐园谈国内的论坛，想做一个优秀的论坛，很难（要不然怎么没有呢）。通看国内大大小小的论坛：那些动机不纯的论坛，上来就想赚大钱，到处转载教程发广告，各种秀节操下线，这些注定要死的； 想做优秀的技术论坛的，比如 CSDN，chinaunix 等，为了迎合中国国情，发帖规则搞的太模糊了。csdn 这么大的一个论坛，作业贴、伸手党不知道占了多少，全是水军。活是活下去了，但是活的怎么样了呢？

我是跟着 CSDN（论坛）、博客园（博客平台）、点点（轻博客）、微博（社交平台）以及后来的知乎一起过来的。博客园在 10 年左右的时候多猛啊，现在呢？微博就不多说了，呵呵！知乎 13 、14年左右的帖子质量多高啊，现在呢？每当我看到「如何评价XXX」 ，「XXX怎样一种体验”」这种典型的知乎式的问题，心里真是一万匹草泥马奔腾啊！

为什么国内做不好这些平台，这么说有点绝对了。那我换个说法，为什么一旦这些平台从「小众」变成「大众」以后就变味了呢？

很多原因（像新浪微博就属于典型的「产品经理的是傻逼」），但是最关键的是：「整体网民素质偏低，而且价值观不正」。你看看在国内的游戏世界里，引导的和崇尚的是什么；再反观魔兽世界，它引导和崇尚的是什么。 国内的大部分游戏从一开始就是为了坑钱而去设计的，他引导玩家去花钱，去炫富，我比砸的钱多，我就完全碾压你，不管你付出多少努力都没用。在知乎的文化里：没约过炮？low B！我给你讲一下我的约炮经历吧，写的和TM小说一样精彩。我们听是宋冬野、我们听的是戴佩妮，我们玩的是小众，小众就真的好吗？抖个机灵，一大堆赞；“泻药，容我先个故事···” 一句话可以解决的问题，你特么写了上万个字，还有一大堆不明觉厉的人点赞。 真正言简意赅且有价值的东西，反而没人理。网民素质偏低的一个体现是：「没有自己的独立判断能力，太容易受环境的引导，缺乏思考」。 基于这个原因，一旦平台的用户量变大了以后，平台没有办法出出力，只能去适应和迎合大众。我从不怀疑知乎团队的坚守和梦想，但是具体能做成什么样的决定权不再他们。

所以，我认为，在国内短期的几年时间里，不可能做出一个高质量的网民交互平台。除非有一天，我们真的开放了，网民素质变高了，有了判断能力。

这三篇文章 技术、论坛、博客、社交 比较枯燥，但是这是对我技术影响很大的一段经历，一定要写出来的。本来打算把 博客和社交 再单拉出来再写一篇，想一想还是算了，这样作为结束也挺好。

关于 Tanky 那段没写好，涉及到人的话写起来总会有一些顾虑，可能会让 Tanky 失望了。
